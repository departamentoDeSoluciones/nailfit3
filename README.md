# Nailfit 💅 (En Construcción 🚧)

Nailfit es una aplicación web diseñada para medir con precisión el tamaño de las uñas utilizando visión por computadora. Actualmente se encuentra en fase de desarrollo activo.

## Inicializar el Proyecto

Para comenzar a trabajar en local, asegúrate de tener [Node.js](https://nodejs.org/) instalado y ejecuta los siguientes comandos en tu terminal:

1. Clona el repositorio.
2. Instala las dependencias del proyecto:

   ```bash
   npm install
   # o usando pnpm: pnpm install
   ```
